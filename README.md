## Installation

1. Clone the repository:
```bash
git clone https://github.com/[your-username]/react-chat-widget.git
```

2. Install dependencies:
```bash
cd react-chat-widget
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Usage

```jsx
import { ChatWidget } from 'react-chat-widget';

function App() {
  return (
    <div>
      <ChatWidget />
    </div>
  );
}
```
